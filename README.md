# 🌾 India Agriculture Crop Production Analysis

## Overview
Exploratory data analysis on India's crop production dataset to uncover patterns across states, crops, and years.

## Questions Answered
1. **Which state produces the most Rice?** → West Bengal
2. **What are the Top 10 crops by total production?** → Coconut dominates by a massive margin
3. **How has Rice production trended over the years?** → Generally increasing from 1997, with a sharp drop in 2020-21

## Key Findings
- Coconut has the highest total production in India by far
- West Bengal is India's largest rice-producing state
- Rice production grew steadily for 2 decades before a 2020-21 decline (likely COVID disruption)

## Tools Used
- Python, Pandas, Matplotlib
- Dataset: India Agriculture Crop Production (Kaggle)
